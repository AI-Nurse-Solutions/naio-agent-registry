# Contributing — NAIO Agent Registry

> Phase 2 (coming soon): the **Nurse-Led Agent Library** — NIN community contributions.

In v1, the registry is curated directly by the NAIO Institute to establish the EDENA-carding standard. Once the standard is proven, NIN members will be invited to submit and steward entries.

## When contributions open, every submission must include:

1. A completed card using [`registry/_TEMPLATE.md`](./registry/_TEMPLATE.md).
2. Verified metadata (license, stars, last commit) — no search snippets.
3. An honest **claim vs. proof** assessment.
4. An assigned **EDENA tier** with justification (see [GOVERNANCE.md](./GOVERNANCE.md)).
5. An explicit **PHI / safety boundary** statement.

## Non-negotiables

- No agent is listed as clinically validated or endorsed.
- No PHI in any example, ever.
- Clinical-risk tools are listed for study with mandatory human-gate language.
- Repos with no license are linked only — never forked or redistributed.

**Agents propose. Humans judge. Nurses steward.**
